# DPS Calculator Setup

## Prerequisites

- Game data unpacked at `../assets/output/` (or set `GAME_DATA_DIR` and `ASSETS_DIR` in `.env`)
- Python 3 with ArknightsDpsCompare at `external/ArknightsDpsCompare` (or pass `--repo <path>`)

## Generate Everything

```bash
# 1. Generate operator_formulas.json (parses Python source)
cargo run --bin generate-dps -- --formulas

# 2. Transpile Python formulas to Rust (~5 min)
cargo run --bin generate-dps -- --transpile

# 3. Generate expected_dps.json (runs Python reference)
cargo run --bin generate-dps -- --expected

# Or all at once:
cargo run --bin generate-dps -- --formulas --transpile --expected
```

## Test

```bash
cargo test --test dps_engine_test -- --no-capture
```

## Environment Variables

| Variable | Default | Description |
|---|---|---|
| `GAME_DATA_DIR` | `../assets/output/gamedata/excel` | Path to game data JSON files |
| `ASSETS_DIR` | `../assets/output` | Path to unpacked assets root |

## Output Files

| File | Generated by |
|---|---|
| `src/dps/config/operator_formulas.json` | `--formulas` |
| `src/dps/custom/generated.rs` | `--transpile` |
| `src/dps/custom/mod.rs` | `--transpile` |
| `tests/fixtures/expected_dps.json` | `--expected` |

**Note**: Ebenholz fails, but that is a Python bug. Rust is correct.
